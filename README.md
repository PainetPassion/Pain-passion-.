# 🥖 Pain et Passion — Site Vitrine

Site vitrine moderne pour la boulangerie **Pain et Passion**, avec animations en séquence d'images et effets visuels premium.

## Structure du projet

```
site-boulange/
├── index.html          # Page principale
├── style.css           # Styles & animations
├── script.js           # Logique & animations JS
└── assets/
    ├── images/         # Images du site (galerie, produits, hero...)
    └── sequence/       # Séquences d'images pour l'animation hero (192 frames)
```

## Hébergement

Ce site est hébergé via **GitHub Pages**.

## Technologies utilisées

- HTML5 / CSS3 (animations, glassmorphism)
- JavaScript vanilla (canvas, effets de parallaxe)
- Séquence d'images animée (192 frames)
